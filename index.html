웹 표준 사이트

웹접근성이란 ?

  1. 장애를 가진 사람들이 웹 콘텐츠를 인지하고,
  2. 편리하게 사용할 수 있으며,
  3. 그 내용이 쉬워야 하며,
  4. 견고성을 지녀야 합니다.

웹 표준 검사

W3C(World Wide Web Consortium) 웹표준 유효성 검사를 했으며 Markup 검증 서비스와 W3C CSS 검사 서비스 모두 통과하였습니다.

스킵 네비게이션 ?

웹접근성과 키보드만으로 접근하는 사용자의 편의를 위해 본문에 스킵 네비게이션을 추가하였습니다.

IR 기법 ?

스크린리더 사용자를 위한 IR(Image Replacement)기법
ir_pm - 의미있는 이미지의 대체 텍스트를 제공하는 경우
ir_wa - 이미지가 없어도 대체 텍스트를 보여주고자 할 때
ir_su - 대체 텍스트가 아닌 접근성을 위한 숨김 텍스트 사용

크로스브라우징

"동일성이 아니라 동등성(등가성)을 의미합니다."
W3C 의 웹 표준 기술에 맞는 코딩을 함으로써 모든 브라우져나 모든 기기에서 사이트가 제대로 보여지고 작동되도록 하는 기법을 말합니다. 본 사이트는 크로스 브라우징 테스트를 마쳤습니다.

반응형 사이트

반응형 사이트란?

pc 외 다양한 디바이스가 등장함에 따라, 사용자의 해상도에 따라 레이아웃을 보다 유연하고 강력하게 설계된 웹사이트를 말합니다. 본 사이트는 미디어쿼리를 이용하여 화면폭을 감지해 다양한 기기에서 최적화된 화면으로 사용자 접근성을 높였습니다.

한줄, 두줄 효과

overflow: hidden;과 text-overflow:ellipsis;로 한줄 효과를
-webkit-box-orient: vertical; -webkit-line-clamp: 2;
를 추가해 여러줄 효과를 주었습니다.

슬라이드 이미지

슬라이드 플러그인을 사용하지 않은 제이쿼리로 직접 상단 배너 이미지 슬라이드 작업을 하였습니다.

라이트 박스

라이트박스 갤러리 플러그인을 통하여 여러장의 앨범 커버를 별도의 팝업창으로 앨범 이미지 정보를 효과적으로 보여주었습니다.

필터 이벤트

포토샵에서 이미지에 필터 효과를 주는 것과 같이 CSS로 블러, 반전, 밝기 조절 등 여러가지 이미지 효과를 주는 것으로 마우스 오버 시 만들 때 각각 이미지에 여러가지 필터 효과를 넣었습니다.

Hover Effect

CSS로 여러가지 스타일의 마우스 오버 효과로 웹 구성을 다이나믹하게 구성했습니다.

                                      (function handleTrailer(){
                                          // 셀럭터 캐시
                                          var $selector = {
                                              body: $( "body" ),
                                              overlay: $( "#blackout" ),
                                              modal: $( "#trailerModal" ),
                                              showButton: $( "#showTrailer" ),
                                              hideButton: $( "#hideTrailer" ),
                                          };
                                          
                                          // 플레이어
                                          var player = {
                                          obj: null, // 플레이어 오브젝트
                                          query : {
                                              theme: "dark",
                                              color: "white",
                                              controls: 1, 
                                              autoplay: 1,
                                              enablejsapi: 1,
                                              modestbranding: 0, // YouTube 로고 감춤
                                              rel: 0,  // 관련 동영상 표시
                                              showinfo: 0, // 제목, 업로더 감춤
                                              iv_load_policy: 3 // 특수효과 감춤
                                          },
                                          visible: false
                                          };
                                          
                                          // 보이기, 숨기기 버튼 활성화
                                          $selector.showButton.on( "click", showPlayer );
                                          $selector.hideButton.on( "click", hidePlayer );
                                          
                                          //YouTube API를 이용해 iframe을 생성
                                          function setPlayer( id ){
                                          player.obj = new YT.Player( "trailer", {
                                              width: 480,
                                              height: 282,
                                              videoId: id,    
                                              playerVars: player.query
                                          });
                                          
                                          // 처음 플레이어 크기 설정
                                          resizePlayer();
                                          
                                          // 리사이즈 화면 회전시 플레이어 크기 다시 설정
                                          $( window ).on( "resize orientationchange", function() {
                                              resizePlayer();
                                          });
                                          }
                                          
                                          function resizePlayer() {
                                              var viewport = {}, frame = {}, modal = {};
                                      
                                              viewport.width = $( window ).width();
                                              viewport.height = $( window ).height();
                                      
                                              frame.width = viewport.width;
                                              frame.height = viewport.width /1.6;
                                      
                                              modal.top = '-100px';
                                              modal.left = "0px";
                                      
                                              $selector.modal.css( modal );
                                      
                                              player.obj.setSize( frame.width, frame.height );
                                          }
                                      
                                          // iframe 보이기
                                          function showPlayer() {
                                              if ( !player.obj ) {
                                                  setPlayer( $selector.showButton.data("youtube") );
                                              }
                                              
                                              $selector.body.addClass("modal_on");
                                              $selector.overlay.show();
                                              player.visible = true;
                                          };
                                          
                                          // iframe 감추기
                                          function hidePlayer (){
                                              player.obj.stopVideo();
                                              $selector.overlay.hide();
                                              $selector.body.removeClass("modal_on");
                                              player.visible = false;
                                          };
                                          
                                      })();
                                      
                                    });
